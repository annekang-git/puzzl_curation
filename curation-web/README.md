# PUZZL Kids Fashion Curation

12-14세 남아를 위한 프리미엄 럭셔리 키즈웨어 큐레이션 웹사이트

## 🎯 기능

- **잡지 스타일 레이아웃**: 여러 레일로 구성된 상품 큐레이션
- **카테고리별 추천**: 상의, 하의, 악세사리(가방), 신상품, 특가 상품
- **반응형 디자인**: 모바일/태블릿/데스크톱 지원
- **상품 클릭 시 puzzl.kr로 이동**

## 📦 레일 구성

1. 🆕 이번 주 신상품
2. 👕 12-14세 남아 인기 상의
3. 👖 12-14세 남아 인기 하의
4. 🎒 키즈 백팩 & 가방
5. 🔥 특가 상품

## 🚀 로컬 개발

```bash
npm install
npm run dev
```

## 🌐 Render 배포

1. Render에서 새 Static Site 생성
2. 이 저장소 연결
3. Build Command: `npm install && npm run build`
4. Publish Directory: `dist`

또는 `render.yaml` 파일을 사용하여 Blueprint 배포

## 🛠 기술 스택

- React 19
- Vite 7
- TailwindCSS 4
- 정적 사이트 (SSG)
